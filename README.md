Если для сервиса вместо localhost необходимо использовать иной хост, то необходимо задать переменную окружения. Если необходимо задать хост, логин и пароль для некоторого подключения rabbitmq, то переменные окружения будет иметь вид RABBITMQ_CONNECTIONS_ИМЯ_ПОДКЛЮЧЕНИЯ_HOST, RABBITMQ_CONNECTIONS_ИМЯ_ПОДКЛЮЧЕНИЯ_USERNAME, RABBITMQ_CONNECTIONS_ИМЯ_ПОДКЛЮЧЕНИЯ_PASSWORD
